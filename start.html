开始使用

安装 Hexo 与「Doku」主题

在 GitHub 上编辑

在使用「Doku」撰写文档之前，你需要先安装 Hexo。

Hexo 是一个基于 Node.js 的高效静态网站生成器（Static Site Generator）。虽然为撰写博客设计，但是可以用于如撰写文档的更多用途。通过 Hexo 你可以轻松地使用 Markdown 撰写内容，除了 Markdown 本身的语法之外，还可以使用 Hexo 提供的「标签插件」来快速的插入特定形式的内容。

安装 Hexo

安装 Hexo 需要 Node.js 和 Git。

你可以参考 Hexo 官方推出的安装指南 安装 Hexo。
